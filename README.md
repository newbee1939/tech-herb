## コンセプト

- あくまで「自分が見たいもの」を一元化して表示することに集中

- Publickeyのように自分の好きな情報を発信するイメージ

## サイトの特徴

- `インプット効率の最大化`
  - このサイトを見たら全てが分かる(技術情報の一元化)
  - なるべくタブ遷移とかリンク遷移はせずに一覧で表示したい
    - 上から下に見ていけばある程度その日の技術トレンドを把握できる
  - 記事の内容を把握するまでの速度を最大化(要約など使って？)
